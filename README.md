## Overview
This repository contains the deliverables for the Smiley Real Estate Data Analyst trial assessment.  
The goal was to analyze sales and agent data across multiple markets and provide actionable insights for marketing and sales. I focused on looking at the bigger picture while completing this assessment, thinking on my feet, staying precise, and delivering quick, actionable results.

## Files Included
- `Data Analyst Trial - Data Set.xlsx` — Original dataset provided for the assessment.  
- `Data Analysis Trial Assessment - Results.pdf` — Final report on Power BI with insights, charts, and recommendations.  
- `best_agents.py` — Python script used to clean and analyze agent-level data, calculate composite scores.  
- `best_markets.py` — Python script used to clean and analyze market-level data, calculate composite scores.  

## How I Analyzed
1. Cleaned and standardized data using Python (pandas).  
2. Calculated composite scores for markets and agents.  
3. Verified data integrity and handled outliers.  
4. Produced charts and summarized actionable insights.  

## Notes & Assumptions
- Composite scores combine key performance metrics.  
- All steps and assumptions are documented in `Data Analysis Trial Assessment - Results.pdf`.
